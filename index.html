O que é?

O YEP é uma plataforma composta por módulos eletrônicos que torna o aprendizado e desenvolvimento de tecnologia mais fácil e divertido.
Voltado para educadores, makers e alunos de STEAM.

O que é?

O YEP é uma plataforma composta por módulos eletrônicos que torna o aprendizado e desenvolvimento de tecnologia mais fácil e divertido.
Voltado para educadores, makers e alunos de STEAM.

Ideal para alunos mais novos

As plaquinhas possibilitam criar uma camada de abstração ao diminuir determinadas complexidades técnicas, ao oferecer blocos de hardware e software previamente desenvolvidos.

Interação com o meio físico

Com elas, é possível interagir com o meio ambiente físico a partir dos sensores e dos atuadores, e assim, controlar coisas, dispositivos, equipamentos e automatizar processos.

Em qualquer linguagem

Os módulos são programados por meio de requisições web que podem ser feitas em qualquer linguagem de programação. O que permite que se consiga trabalhar com sua linguagem de programação favorita.

Programado de maneira sem fio

Por conta das plaquinhas serem programadas via requisições web, os módulos do YEP são sem fio. Sem confusão com jumpers e protoboards.

Para quem?

Você é estudante?

O YEP é uma excelente ferramenta para estudantes, sobretudo, para que está começando.

A plataforma é modular e sem fio, permitindo que os alunos explorem sua criatividade e desenvolvam lógica de programação de uma maneira mais divertida.

Você é professor?

As plaquinhas do YEP podem ser utilizadas por professores durante suas aulas.

Com elas, os professores podem escolher trabalhar dentro da sua linguagem de programação de domínio ou com sua ferramenta favorita.

Você é maker ?

Os módulos do YEP são uma ótima opção para os makers

Com eles, os makers conseguem focar mais na aplicação e evitar problemas de eletrônica e de hardware. Também é uma excelente opção para desenvolvedores de software que queiram se aventurar em IoT.
